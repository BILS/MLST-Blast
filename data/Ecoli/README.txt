Data files downloaded 12-23-2014 from http://pubmlst.org/data/dbases.xml.

Fasta files and profiles are saved in folder Ecoli.

Species names and acronyms are given in Ecoli/SPECIES.txt.

Fasta file blastdb.fas for blast was created in Ecoli/BLASTDB.

A PROFILES.txt with all ST-profiles was saved in Ecoli/PROFILES.

Note that dashes in the fasta headers have been replaced with underscores for some files.
